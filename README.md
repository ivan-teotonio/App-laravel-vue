<p dir="auto"><a href="https://wmazoni-sds1.netlify.app" rel="nofollow">https://wmazoni-sds1.netlify.app</a></p>
<p dir="auto">Big Game Survey é uma aplicação full stack web e mobile construída durante a 1ª edição da <strong>Semana DevSuperior</strong> (#sds1), evento organizado pela <a href="https://devsuperior.com" title="Site da DevSuperior" rel="nofollow">DevSuperior</a>.</p>
<p dir="auto">A aplicação consiste em uma pesquisa de preferência de games, onde os dados são coletados no app mobile, e depois são listados no app web, que também apresenta um dashboard com gráficos baseados nestes dados.</p>
